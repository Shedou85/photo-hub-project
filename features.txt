 Cloudflare R2 Setup Guide  [ATLIKTA 2026-02-27 — R2 pilnai integruotas ir veikia produkcijoje]                                                                                                                                                                                                   
  1. Cloudflare paskyra                                                                                                                                                                                                    
  
  Jei dar neturi — eik į https://dash.cloudflare.com/sign-up ir susikurk nemokamą paskyrą.

  2. Įjungti R2

  1. Prisijungęs į Cloudflare Dashboard, kairėje šoninėje juostoje rask "R2 Object Storage"
  2. Paspaudus paprašys pridėti payment method (kredito kortelę) — tai reikalavimas net nemokamai plano, bet nieko nemokamai neapmokestins kol neviršysi free tier
  3. Patvirtink ir R2 bus aktyvuotas

  3. Sukurti Bucket

  1. R2 puslapyje spausk "Create bucket"
  2. Bucket name: photohub-photos (arba pixelforge-photos)
  3. Location: pasirink EU (Europe) — arčiausiai tavo vartotojų
  4. Storage class: palik default (Standard)
  5. Spausk "Create bucket"

  4. Sugeneruoti API Keys

  Tai svarbiausia dalis — šie raktai bus naudojami PHP backend'e:

  1. R2 puslapyje spausk "Manage R2 API Tokens" (viršuje dešinėje)
  2. Spausk "Create API Token"
  3. Nustatymai:
    - Token name: photohub-backend
    - Permissions: Object Read & Write
    - Specify bucket(s): pasirink tavo sukurtą bucket'ą (photohub-photos)
    - TTL: palik "Forever" (arba nustatyk pagal poreikį)
  4. Spausk "Create API Token"
  5. SVARBU — iškart nusikopijuok šiuos duomenis (rodomi tik vieną kartą):

  Access Key ID:     xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  Secret Access Key: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

  5. Surinkti visus reikalingus duomenis

  Po setup'o turėsi šiuos duomenis (reikės backend integracijai):

  ┌───────────────────┬─────────────────────────────────────────────────────────────┬─────────────────┐
  │    Parametras     │                          Kur rasti                          │    Pavyzdys     │
  ├───────────────────┼─────────────────────────────────────────────────────────────┼─────────────────┤
  │ Account ID        │ Cloudflare Dashboard → dešinėje šoninėje                    │ a1b2c3d4e5f6... │
  ├───────────────────┼─────────────────────────────────────────────────────────────┼─────────────────┤
  │ Access Key ID     │ Iš 4 žingsnio                                               │ xxxxxxxx...     │
  ├───────────────────┼─────────────────────────────────────────────────────────────┼─────────────────┤
  │ Secret Access Key │ Iš 4 žingsnio                                               │ xxxxxxxx...     │
  ├───────────────────┼─────────────────────────────────────────────────────────────┼─────────────────┤
  │ Bucket Name       │ Ką pavadinai 3 žingsnyje                                    │ photohub-photos │
  ├───────────────────┼─────────────────────────────────────────────────────────────┼─────────────────┤
  │ Endpoint URL      │ Automatiškai: https://<ACCOUNT_ID>.r2.cloudflarestorage.com │                 │
  └───────────────────┴─────────────────────────────────────────────────────────────┴─────────────────┘

  Account ID rasiesi Cloudflare Dashboard — bet kurio domeno puslapyje dešinėje arba R2 bucket'o settings puslapyje.

  6. (Pasirinktinai) Public Access

  Jei norėsi, kad nuotraukos būtų pasiekiamos per URL tiesiogiai (pvz. klientų peržiūrai):

  1. Eik į savo bucket'ą → "Settings"
  2. "Public access" → "Allow Access"
  3. Gali priskirti custom domain (pvz. photos.pixelforge.pro) arba naudoti Cloudflare suteiktą *.r2.dev subdomeną
